Usage Rules
01. 새 페이지 → 이 문서에서 클래스 복사. 새 클래스 창시 금지.
02. 변형 필요 → BEM 수식어 패턴: .card--[variant]
03. 색상 직접 입력 금지 → var(--*) 토큰만 사용
04. 버튼 텍스트 → 한/영 혼용 가능. uppercase는 영문에만.
05. 간격 → var(--space-*) 만 사용. 임의 px 값 금지.
06. 새 컴포넌트 패턴 필요 시 → 이 파일 + style.css 동시 업데이트.
