# Kinfolk — 레퍼런스 브랜드

> **참고 축**: 편집 톤, 여백의 미학, 슬로우 라이프 미디어
> **WOOHWAHAE와의 거리**: 가장 가까운 편집 감각의 선례

---

## 핵심 특성

- **매체**: 독립 라이프스타일 매거진 (2011~, Portland → Copenhagen)
- **톤**: 절제와 관조. 호들갑스러운 감탄사를 배제하고 여백으로 현상을 관측한다.
- **시각**: 모노크롬 기조, 여백 60% 이상 확보, 조작 없는 자연광 중심 포토그래피
- **구조**: 특정한 테마 단위의 Issue 발행 (계절/주제별 깊이 우선)
- **독자 관계**: 억지로 설득하지 않는다. 독자가 스스로 발견하게 둔다.

## WOOHWAHAE가 취할 것

- 현상을 쪼개지 않고 깊이 파고드는 Issue 기반 매거진 구조
- 여백과 침묵을 시각적/언어적 편집 도구로 구사하는 감각
- "적게 말하되 깊게 꽂히는" 언어의 질량 관리

## WOOHWAHAE가 다른 점

- Kinfolk은 서구 슬로우 라이프. WOOHWAHAE는 동아시아 수행 문화가 뿌리.
- Kinfolk은 미디어 중심. WOOHWAHAE는 시스템(LAYER OS) + 아틀리에(실천) 통합.
- Kinfolk은 관찰자 시점. WOOHWAHAE는 실천자(순호)의 1인칭 기록.
