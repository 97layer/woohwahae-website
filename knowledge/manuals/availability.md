# System Availability & Power Management

**"노트북을 닫으면 97layerOS도 잠듭니다."**

현재 귀하의 시스템(97layerOS)은 MacBook 로컬 환경에서 구동되고 있습니다. 노트북 덮개를 닫거나 절전 모드에 진입하면 CPU와 네트워크가 차단되므로 텔레그램 데몬도 멈추게 됩니다.

## 해결 방법 (3가지 옵션)

### 1. Clamshell Mode (로컬 유지 / 비용 0원)

맥북을 덮어두고도 계속 실행하려면 **"클램쉘 모드"** 조건을 충족해야 합니다.

- **조건**: 전원 케이블 연결 + 외부 모니터 연결 + 외부 키보드/마우스 연결.
- **장점**: 추가 비용 없이 현재 환경 그대로 사용.
- **단점**: 장비 연결이 필수.

### 2. Prevent Sleep App (로컬 유지 / SW적 해결)

강제로 잠들기를 방지하는 앱을 사용합니다.

- **추천**: `Amphetamine` (무료, App Store) 또는 터미널 명령어 `caffeinate`.
- **방법**: `caffeinate -i -s` 명령어로 데몬을 감싸서 실행 (화면은 꺼져도 시스템은 유지).
- **주의**: 배터리 소모가 극심하며, 이동 중 가방 안에서 발열 위험 있음.

### 3. Cloud Deployment (진정한 서버화 / 월 5,000원~)

시스템을 클라우드 서버(AWS, Vultr, Fly.io 등)로 이주합니다.

- **장점**: 노트북이 꺼져 있어도 24/7 365일 무중단 가동.
- **단점**: 소액의 서버 비용 발생 & 초기 세팅 필요.
- **추천**: 97layerOS의 안정성을 위해 궁극적으로 가야 할 방향.

---
**Technical Note**:
현재 적용된 `launchd` 나 `caffeinate` 명령어라도, 맥북의 하드웨어적 Sleep(Lid Close without Power)을 이길 수는 없습니다. **24시간 비서**를 원하신다면 **옵션 3(클라우드)** 을 권장합니다.
