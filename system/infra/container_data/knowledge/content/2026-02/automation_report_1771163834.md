**임무 수행 절차:**

1.  파일 내용을 제공받습니다. (파일을 직접 열어볼 수 없으므로, 파일 내용을 텍스트로 제공해 주시면 됩니다.)
2.  파일 내용을 분석합니다.
3.  분석 결과를 바탕으로 브랜드 전략 인사이트를 도출합니다.
4.  도출된 인사이트를 보고서 형식으로 제공합니다.

파일 내용을 알려주시면 즉시 분석을 시작하여 브랜드 전략 인사이트를 도출하겠습니다.

## CE Draft
알겠습니다. 97layerOS의 전략 분석가로서 제공해주신 분석 결과를 바탕으로 브랜드 블로그/포스트 초안을 작성하겠습니다. 분석 결과는 다음과 같은 형식으로 제공해주시면 됩니다.

*   **핵심 내용 요약:** 분석 결과의 핵심 내용을 3-5 문장으로 요약합니다.
*   **강점 (Strengths):** 브랜드의 주요 강점을 3-5가지 제시합니다.
*   **약점 (Weaknesses):** 브랜드의 주요 약점을 3-5가지 제시합니다.
*   **기회 (Opportunities):** 브랜드가 활용할 수 있는 시장 기회를 3-5가지 제시합니다.
*   **위협 (Threats):** 브랜드가 직면한 위협 요소를 3-5가지 제시합니다.
*   **타겟 고객:** 브랜드의 주요 타겟 고객층을 정의합니다. (인구 통계, 관심사, 행동 양식 등)
*   **핵심 메시지:** 타겟 고객에게 전달할 핵심 메시지를 2-3 문장으로 요약합니다.
*   **권장 사항:** 브랜드 전략 실행을 위한 구체적인 권장 사항을 3-5가지 제시합니다.

위 정보를 제공해주시면, 이를 바탕으로 매력적인 브랜드 블로그/포스트 초안을 작성하여 제공해 드리겠습니다.
