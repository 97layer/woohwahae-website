# /brand — Brand OS 참조

Brand OS 문서를 빠르게 조회한다.

## 실행 순서

1. **Brand 디렉토리 인덱스**
```bash
cat directives/practice.md | head -50
```

2. **핵심 문서 요약 출력**

| 파일 | 내용 |
|------|------|
| README.md | 전체 구조 안내 |
| experience_map.md | 세션 구조, 접점 설계 (E+M) |
| foundation.md | 문서 레이어 구조 (F) |
| story.md | 개인 서사 (G) — FROZEN |
| audience.md | 타겟, 포지셔닝, 시나리오 (H+P+U) |
| content_system.md | Archive/Magazine, Hook-Story-Core-Echo (J) |
| design_tokens.md | 디자인 시스템 (K+V) |
| voice_tone.md | 톤, 채널별 규칙 (L) |
| service_ritual.md | 아틀리에 6단계 플로우 (R) |
| teaching.md | B2B/교육 (T) |
| roadmap.md | 비전, 성과 기준, 커뮤니티 (N+Y) |

3. **IDENTITY.md 핵심 선언**
```bash
head -30 directives/the_origin.md
```

## 사용법
- 에세이 작성 시 → `content_system.md` + `voice_tone.md` 참조
- 디자인 검증 시 → `design_tokens.md` 참조
- 브랜드 스코어 시 → `the_origin.md` + `CD.md` 참조
