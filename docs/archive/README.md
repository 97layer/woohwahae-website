# Docs Archive

Historical notes live here when they are still worth keeping for continuity but
should not remain in the default Layer OS read path.

Rules:

- archive notes are not active runtime authority
- archive notes may preserve rollout history, temporary strategy, or extracted legacy thinking
- active contracts still live in `AGENTS.md`, `constitution/`, `contracts/`, and the current canonical docs under `docs/`
- when an archive note becomes unnecessary, delete it instead of letting it drift back into active guidance
