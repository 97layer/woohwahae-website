# Kinfolk — 레퍼런스 브랜드

> **참고 축**: 편집 톤, 여백의 미학, 슬로우 라이프 미디어
> **WOOHWAHAE와의 거리**: 가장 가까운 편집 감각의 선례

---

## 핵심 특성

- **매체**: 독립 라이프스타일 매거진 (2011~, Portland → Copenhagen)
- **톤**: 절제된, 관조적, 사색적. 감탄사 없음. 여백이 말함.
- **시각**: 모노크롬 기조, 여백 60%+, 자연광 중심 포토그래피
- **구조**: 테마 기반 Issue 발행 (계절/주제 단위)
- **독자 관계**: 설득하지 않음. 발견하게 함.

## WOOHWAHAE가 가져갈 것

- Issue 기반 매거진 발행 구조
- 여백과 침묵을 편집 도구로 사용하는 감각
- "적게 말하되 깊이 닿는" 언어 밀도

## WOOHWAHAE가 다른 점

- Kinfolk은 서구 슬로우 라이프. WOOHWAHAE는 동아시아 수행 문화가 뿌리.
- Kinfolk은 미디어 중심. WOOHWAHAE는 시스템(LAYER OS) + 아틀리에(실천) 통합.
- Kinfolk은 관찰자 시점. WOOHWAHAE는 실천자(순호)의 1인칭 기록.
