## 1. 어조 스펙트럼

WOOHWAHAE의 목소리는 하나가 아니다. 채널에 따라 스펙트럼 위에서 이동한다.

```
[사색적] ──────────────────────── [실용적]
  Archive    About    Service    Shop
  에세이     철학선언   안내       제품설명
```

공통: 조용하다. 밀도 있다. 큰 소리 내지 않아도 들리는 사람에게 닿는다.

## 6. 문장 규칙

- **캡션**: 50-100단어. 과잉 설명 금지.
- **에세이**: 5000-8000자. Hook-Story-Core-Echo 구조 (→ content_system.md).
- **한 문장이 전부를 말할 수 있어야 한다.**
- 해시태그 금지. 이모지 금지. 클릭베이트 금지.
- 불확실하면 빈 칸으로 둔다. 채우지 않는다.

---

## 7. 감탄사/빈 공감 금지

에이전트 산출물에서 절대 사용 금지:
- "오!", "훌륭합니다", "대단합니다"
- "좋은 생각이네요", "멋지군요"
- 불필요한 수식어, 과잉 형용사

## 8. 이중 모드 웹 카피 — 짜임새 vs 담백

웹 카피는 두 가지 모드로 존재한다. 섹션 성격에 따라 선택한다.

> **판단 기준**: 이 카피가 브랜드 세계관을 *선언*하는가? → 짜임새
> 이 카피가 사용자에게 *정보*를 전달하는가? → 담백

### 짜임새 (Crafted) — 선언, 철학, About/Archive

세계관을 밀도 있게 표현한다. 한 문장이 전부를 말해야 한다.

**규칙** (C1-C6):
- C1: 동사보다 명사. "제공합니다" ❌ → "기록이다" ✅
- C2: 단정적 선언. 의문문/청유문 제한
- C3: 허용 키워드 20개 우선 사용 (§3 참조)
- C4: 한 문장 = 하나의 생각. 접속사로 연결하지 않는다
- C5: 여백을 문장 사이에 허용 — 침묵도 리듬
- C6: 동아시아 감수성 — 추상에서 구체로, 구체에서 본질로

**예시**:
```
❌ "저희는 슬로우라이프를 추구하는 헤어 아틀리에입니다."
✅ "헤어에서 시작해 일상의 리듬을 다시 설계한다."

❌ "여러분의 아침을 더 여유롭게 만들어 드립니다."
✅ "아침 30초. 그게 하루의 첫 번째 의식이다."
```

### 담백 (Plain) — 안내, 서비스, 예약/연락

필요한 정보만. 수식 없이.

**규칙** (P1-P6):
- P1: 합니다체. 간결하고 정중.
- P2: 주어-술어 직결. 수식어 최소화
- P3: 숫자/시간/조건은 명확하게 표기
- P4: 감탄사, 친근체, 이모지 금지
- P5: 한 문장 = 하나의 정보
- P6: 모호하면 빈 칸. 채우지 않는다

**예시**:
```
❌ "편안한 상담을 위해 미리 연락 주세요~^^"
✅ "예약은 3일 전까지 가능합니다."

❌ "저희 아틀리에를 찾아주시면 최고의 서비스를 제공하겠습니다!"
✅ "예약 후 방문 시 상담을 진행합니다."
```

### 섹션별 모드 맵

| 섹션 | 모드 | 이유 |
|------|------|------|
| About manifesto | 짜임새 | 철학 선언 |
| Archive 에세이 리드 | 짜임새 | 브랜드 세계관 |
| Archive TOC 캡션 | 짜임새 | 독자 유입 |
| Service 소개 | 짜임새 (리드) + 담백 (상세) | 혼합 |
| 예약 안내, 운영 시간 | 담백 | 정보 전달 |
| Offering 페이지 | 담백 (서비스 내용) + 짜임새 (도입부) | 혼합 |
| Contact, 404 | 담백 | 기능적 |
