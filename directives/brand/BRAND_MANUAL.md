### 2D. Surface

**Film Grain** — 전체 화면에 항상 적용. "날것의 질감."
```css
body::after {
  position: fixed; inset: 0; z-index: 9999; pointer-events: none;
  opacity: 0.035;
  background-image: SVG feTurbulence(baseFrequency='0.75', numOctaves='4');
  mix-blend-mode: multiply;
}
```
변형: `home-radical::after` → `opacity: 0.05`, `baseFrequency: 0.85`
카드 이미지: `baseFrequency: 0.9`, `opacity: 0.3`, `blend: overlay` → hover 시 `opacity: 0.1`

**Glassmorphism** — nav 스크롤, 모바일 오버레이, 아티클 플로팅 nav:
```css
--glass-surface: rgba(255,255,255,0);    /* 순수 블러만 */
--glass-blur:    saturate(180%) blur(20px);
--glass-border:  rgba(0,0,0,0.05);
/* box-shadow: 0 4px 30px rgba(0,0,0,0.03) */
```

**Selection** — 텍스트 선택 시 브랜드 감각:

### 2E. Motion

**Breath System** — 인간 호흡 ~0.27Hz (주기 3.8s) 기반:

| Token | 값 | 용도 |
|-------|----|------|
| `--breath` | `3.8s` | 한 호흡 주기 (낮 기준) |
| `--breath-half` | `1.9s` | 반 호흡 |
| `--breath-quarter` | `0.95s` | 1/4 호흡 (fb 전환) |

시간대 변조: → 2A. Color 테이블 참조.

호흡 적용 예: hero 동심원 pulse, manifesto opacity(1↔0.94), fb-statement/fb-brand opacity(1↔0.82)

**Easing Curves**:

| Token | 값 | 성격 |
|-------|----|------|
| `--ease` / `--ease-out` | `cubic-bezier(0.16, 1, 0.3, 1)` | Apple-style ease-out-expo. 내쉬는 곡선 |
| `--ease-in-out` | `cubic-bezier(0.42, 0, 0.58, 1)` | 들이쉬고 내쉬는 곡선 |
| `--ease-wave` | `cubic-bezier(0.37, 0, 0.63, 1)` | 정현파 근사 S곡선 |

**Duration**:
- `--duration`: `0.6s` — 기본 전환
- `--duration-slow`: `1.2s` — 히어로/핵심 요소 (시간대별 1.2~2.0s 변조)

**Stagger Pattern** — 순차 요소 리빌:

| 컴포넌트 | 간격 | 수학적 근거 |
|----------|------|-------------|
| TOC row | `70ms` | ~1/54 breath |
| Archive card | `60ms` | ~1/63 breath |
| Nav links (fb--nav) | `62ms` | 1/16 breath-quarter |
| Article header | `0 / 80ms / 150ms` | 계단식 |
| Hero ring | `300ms` | 입장 stagger, 0.2s/0.5s/0.8s/1.1s |

모바일(≤768px): archive card stagger → `0s` (단일 컬럼이므로 해제)

**Scroll Reveal**:
```css
.fade-in { opacity: 0; transform: translateY(20px) scale(0.985); }
.fade-in.visible { opacity: 1; transform: translateY(0) scale(1); transition: 0.8s var(--ease); }
.fade-in-slow { translateY(30px) scale(0.98); transition: 1.2s; }  /* 히어로/핵심 */
```
IntersectionObserver 연동. 형제 순서 기반 80ms 자동 stagger.

**Scroll-Driven Animation** (Chrome 115+ / Safari 26+):
```css
/* hero-drift: 스크롤 0~45vh에서 opacity 1→0, translateY 0→-30px */
animation-timeline: scroll(root); animation-range: 0vh 45vh;
/* progress-grow: 우측 2px 세로선 fill 0→100% */
animation-timeline: scroll(root); animation-range: 0% 100%;
```
미지원 브라우저: JS fallback (`transition: height 0.1s linear`).

**Hero SVG Animation**:
```
동심원 4개: 순차 fade-in (0.2s 간격) → opacity 0.45 → pulse (0.30↔0.55)
나선(spiral): stroke-dashoffset 2200→0 (3.6s draw) → breathe (0.6↔0.85)
중심점: appear 0.5s + pulse scale(1↔1.8, 4s)
ECG 파형: fade-in 1.2s + shimmer opacity(0.30↔0.45)
```
`animation-play-state: paused` → `.hero-show` 클래스 추가 시 `running`으로 전환.

## 5. Service & Teaching

### 5A. Service — Wash & Go

헤어 아틀리에는 기술을 파는 공간이 아니다. 사람을 만나는 공간이다.

**Wash & Go**: 매일 아침 30초로 완성되는 헤어. 과도한 유지보수 = 그 사람의 시간을 빼앗는 것.

**핵심 질문**: "어떤 헤어를 원하세요?" ❌ → "아침에 헤어에 몇 분 쓰시나요?" ✅

**Client Journey 7단계**:

| 단계 | 이름 | 핵심 |
|------|------|------|
| 1 | Discovery | 콘텐츠/소개로 아틀리에 발견 |
| 2 | Exploration | 웹사이트에서 철학과 서비스 탐색 |
| 3 | Booking | 예약 (네이버/DM/전화) |
| 4 | Consultation | 상담 — 라이프스타일 파악, 스타일 제안 |
| 5 | Service | 시술 — 과정을 설명하며 진행 |
| 6 | Completion | 마무리 — 홈케어 가이드 (간결하게) |
| 7 | Continuation | 재방문은 자연스럽게. 압박 금지 |

**서비스 금지**:
- ❌ 트렌드 스타일 강요
- ❌ "이번 시즌 유행" 언급
- ❌ 과도한 제품 추천
- ❌ 빈도 높은 재방문 압박
- ❌ 후기 요청

### 7A. Do / Don't 통합

**DO** ✅:
- 여백 60%+ 유지
- Monochrome 기본, 컬러는 이유 있을 때만
- 허용 키워드 20개 내에서 표현

- 개인적인 것 → 보편적인 것으로 확장
- 불완전하지만 진심인 것을 택하기
- 과정을 공개하기

| 영역 | 금지 |

| **서비스** | 트렌드 강요 / "이번 시즌" / 재방문 압박 / 후기 요청 / 과잉 제품 추천 |
