# Junction Protocol

97layer (개인 경험) ↔ WOOHWAHAE (브랜드 자산) 접합 프로토콜.
개인의 파편을 브랜드의 자산으로 전환하는 시스템.

---

## I. 프로토콜 정의

**Junction (접합)**이란:
- 97layer의 개인적 경험/사유를 WOOHWAHAE 브랜드 콘텐츠로 전환하는 과정
- 일기 → 브랜드 자산
- 사적 → 공적
- 파편 → 서사

**목적**:
1. 개인 경험의 휘발 방지
2. 진정성 있는 브랜드 콘텐츠 생성
3. 97layer ↔ WOOHWAHAE 일관성 유지
4. 브랜딩 일기 시스템 구축

---

## II. 1:1:1:1 접합 맵

### 전체 구조

```
97layer (내면) → WOOHWAHAE (철학) → Visual (상징) → Aesop (벤치마크)
```

이 4개 레이어는 항상 일치해야 한다.

### 상세 매핑

| 97layer | WOOHWAHAE | Visual | Aesop |
|---------|-----------|--------|-------|
| 고독한 본질 탐구 | Slow (자신의 파동) | 여백 많은 구도 | 긴 텍스트 허용 |
| 본질 집착 | 실용적 미학 | 블런트 상징 | 제품 본질 강조 |
| 침묵의 가치 | 무언의 교감 | 모노크롬 | 절제된 언어 |
| 외부 평가 거부 | 자기 긍정 | 불완전 허용 | 트렌드 무시 |
| 시간 기록 강박 | 아카이브 | 인장 상징 | 시간 축적 |
| 불완전 수용 | 과정 중시 | 나비 (변화) | Becoming |
| 선례 남기기 | 삶의 기록 | 사용 흔적 | 이야기 중시 |
| 알고리즘 비판 | 반-휘발 | 세리프 타이포 | 느린 읽기 |

---

## III. 브랜딩 일기 시스템

### 작동 방식

```
1. Capture (포착)
   ↓
2. Connect (연결) - 과거 경험과 연결 발견
   ↓
3. Meaning (의미) - 개인 → 보편 전환
   ↓
4. Edit (편집) - 브랜드 톤 적용
   ↓
5. Publish (발행) - WOOHWAHAE 콘텐츠화
```

### 단계별 상세

#### 1단계: Capture (SA 에이전트)

**입력**:
- 텔레그램 메시지 (일상 기록)
- 외장하드 스캔 (영상/사진)
- 메모 파일 (단상)
- 음성 녹음 (사유 과정)

**저장 위치**:
- `knowledge/raw_signals/` (가공 전)
- 태그: 날짜, 감정, 키워드

**원칙**:
- 필터링 없이 모두 저장
- 불완전해도 OK
- 맥락 보존 (전후 문장)

#### 2단계: Connect (SA 에이전트)

**작업**:
- 과거 기록과 유사성 발견
- 반복되는 테마 식별
- 철학과의 연결 고리 찾기

**출력**:
- 연결 그래프 (`knowledge/connections.json`)
- "이 경험은 [철학]과 연결됨"

**예시**:
```
입력: "오늘 미용실 청소하면서 10년 전 반지하 생각남"
연결:
- 97layer_identity.md → 시간 아키비스트
- WOOHWAHAE → 아카이브 철학
- 콘텐츠 가능성: "공간과 시간" 에세이
```

#### 3단계: Meaning (CE 에이전트)

**작업**:
- 개인적 → 보편적 전환
- "나만의 이야기"를 "우리의 이야기"로
- 철학적 언어로 재진술

**변환 예시**:

**Before** (개인):
> "오늘 손님 없어서 하루종일 책만 읽었다. 매출은 0원이지만 나쁘지 않았다."

**After** (보편):
> "비어있는 공간이 주는 선물이 있다. 아무도 찾지 않는 날, 미용실은 연구실이 된다. 숫자로 표현할 수 없는 것들에 시간을 쓰는 것. 그것이 본질을 지키는 방법이다."

#### 4단계: Edit (CE 에이전트)

**작업**:
- WOOHWAHAE Tone & Manner 적용
- 서사 구조 적용 (The Hook / The Manuscript / The Afterglow)
- 시각 요소 제안 (이미지/영상)

**체크리스트**:
- [ ] 5가지 철학 중 1개 이상 연결
- [ ] Aesop 벤치마크 톤 유지
- [ ] 과장/감정 과잉 제거
- [ ] 질문으로 끝남
- [ ] 400-800자 (인스타그램 기준)

#### 5단계: Publish (CD 승인)

**발행 전 검토** (CD 에이전트):
- 브랜드 일관성 체크
- 시각 요소 적절성
- 발행 타이밍 (요일/시간)

**발행 채널**:
- 인스타그램 (Primary)
- 웹사이트 아카이브 (Future)
- 뉴스레터 (Future)

**발행 후**:
- `knowledge/assets/published/` 이동
- 메타데이터 기록 (날짜, 반응, 태그)
- 다음 콘텐츠 아이디어 생성

## IV. 경험 카테고리별 접합 방법

### 카테고리 1: 미용사 경험

**예시 경험**:
- 고객과의 대화
- 시술 중 깨달음
- 공간 운영 고민
- 10년 커리어 회고

**접합 방향**:
→ **실용적 미학** 또는 **무언의 교감**

**콘텐츠 형식**:
- 에세이 (800자)
- 공간 사진 + 캡션
- 고객 후기 (익명)

**금지**:
- Before/After 과장
- 시술 자랑
- 가격 언급

### 카테고리 2: 고독한 사유

**예시 경험**:
- 혼자 있는 시간
- 인문학 독서
- 내면 탐구
- 철학적 질문

**접합 방향**:
→ **Slow (자신의 파동)**

**콘텐츠 형식**:
- 텍스트 카드 (The Hook만)
- 긴 캐러셀 (10장)
- 롱폼 영상 (10분)

**금지**:
- 가벼운 톤
- 빠른 소비 유도
- 답 제시

### 카테고리 3: 시간과 기록

**예시 경험**:
- 과거 회상
- 외장하드 정리
- 공간 정리
- 브랜딩 일기 작성

**접합 방향**:
→ **아카이브**

**콘텐츠 형식**:
- 타임라인 시리즈
- 과거 사진 + 현재 해석
- "N년 전 오늘" 형식

**금지**:
- 과거 미화
- 성공 스토리 과장
- 향수 과잉

### 카테고리 4: 알고리즘 비판

**예시 경험**:
- SNS 피로감
- 자극적 콘텐츠 비판
- 생산성 압박 거부
- 트렌드 추종 거부

**접합 방향**:
→ **Slow** + **자기 긍정**

**콘텐츠 형식**:
- 비판 에세이 (신중하게)
- 대안 제시 (WOOHWAHAE 방식)
- "하지 않기" 리스트

**금지**:
- 공격적 톤
- 특정 브랜드 저격
- 우월감 표현

### 카테고리 5: 불완전과 실패

**예시 경험**:
- 완성 못한 프로젝트
- 실패한 시도
- 완벽주의 마비
- 지운 글/영상

**접합 방향**:
→ **과정 중시** (羽化 메타포)

**콘텐츠 형식**:
- "불완전한 기록" 시리즈
- 과정 공개
- 실패담 (교훈 아닌 기록)

**금지**:
- 자책 과잉
- 성장 스토리로 포장
- "이렇게 극복했다" 결말

## V. 접합 불가능 경험

모든 경험이 브랜드 자산이 되는 것은 아니다.

### 제외 기준

**❌ 정치적 발언**:
- 97layer 개인 의견이지만 브랜드화 부적절
- 예외: 보편적 가치 (인권, 환경 등)

**❌ 사적 갈등**:
- 특정인과의 갈등
- 가족 문제
- 금전 문제

**❌ 상업적 내용**:
- 매출 자랑
- 제품 홍보
- 할인 이벤트

**❌ 감정 과잉**:
- 과도한 분노
- 신파조
- 자기 연민

### 보관 방법

제외된 경험도 삭제하지 않고:
- `knowledge/private/` 폴더에 보관
- 에이전트 접근 불가 설정
- 향후 재평가 가능

- [ ] 읽는 시간 3-5분
- [ ] 질문으로 여운 남김

- [ ] 타인 비방 없음

## VII. 발행 주기 및 타이밍

### 정기 발행

**인스타그램**:
- 주 1-2회 (월요일, 목요일)
- 시간: 오전 10시 또는 오후 8시
- 계절별 조정 가능

**이유**:
- 일관성 (알고리즘 아닌 습관)
- 완벽주의 방지 ("이번 주는 내야 함")
- 아카이브 축적

### 비정기 발행

**조건**:
- 긴급한 사유 (특별한 경험)
- 계절적 타이밍 (첫눈, 벚꽃 등)
- 협업 콘텐츠

**원칙**:
- 정기 발행 우선
- 비정기는 월 1회 이하

## VIII. 에이전트별 역할

### SA (Strategy Analyst)

**역할**: Capture + Connect
- 텔레그램 메시지 자동 수집
- 외장하드 스캔 (주 1회)
- 연결 그래프 생성
- 콘텐츠 가능성 제안

**출력**:
- `knowledge/raw_signals/` 저장
- `knowledge/connections.json` 업데이트
- CE에게 "콘텐츠 후보" 리스트 전달

### CE (Chief Editor)

**역할**: Meaning + Edit
- 개인 → 보편 전환
- 서사 구조 적용
- 초고 작성 (400-800자)
- 시각 요소 제안

**출력**:
- `knowledge/assets/draft/` 저장
- CD에게 검토 요청

### CD (Creative Director)

**역할**: Publish 승인
- 브랜드 일관성 최종 체크
- 발행 타이밍 결정
- 거부 시 수정 방향 제시

**권한**:
- 발행 거부 가능
- 수정 요청 가능
- 보류 결정 (타이밍 조정)

### AD (Art Director)

**역할**: 시각화
- CE 제안 받아 이미지 선택
- 타이포그래피 적용
- 인스타그램 피드 구성
- 최종 시안 제작

**출력**:
- 발행 가능한 이미지/영상
- 캡션 최종본

### TD (Technical Director)

**역할**: 시스템 유지
- 파일 경로 관리
- 메타데이터 기록
- 백업 (외장하드)
- 자동화 스크립트 작성

## IX. 실제 사례 (예시)

### 사례 1: "공간 정리" 경험

**1. Capture (SA)**:
```
입력 (텔레그램): "외장하드 정리했다. 2년 전 영상 발견.
결국 편집 안 했네. 시간만 흘렀다."
```

**2. Connect (SA)**:
```
연결:
- 97layer_identity.md → 시간 아키비스트, 휘발의 반복
- WOOHWAHAE → 아카이브 철학
- 과거 콘텐츠: "불완전한 기록" (2024.11)
→ 콘텐츠 가능성: 높음
```

**3. Meaning (CE)**:
```
개인 → 보편 전환:
"외장하드는 현대인의 타임캡슐이다.
우리는 기록하지만 돌아보지 않는다.
파일명에 날짜만 적힌 채, 시간은 흘러간다."
```

**4. Edit (CE)**:
```
The Hook:
"공간을 정리한다는 것은 시간을 정리하는 일이다"

The Manuscript:
외장하드 속 영상들. 2년 전, 3년 전, 어떤 것은 5년 전.
파일명: "201912_테스트", "20200304_초안", "편집필요_최종".
열어보지 않아도 안다. 결국 완성하지 못했다는 것을.

우리는 끊임없이 기록한다.
하지만 돌아보지 않는다.
파일은 쌓이고, 용량은 차고, 시간은 흐른다.

공간 정리는 시간 정리다.
무엇을 남기고, 무엇을 지울 것인가.
완성하지 못한 것들을 어떻게 받아들일 것인가.

불완전한 기록들이 모여 한 사람의 초상이 된다.
매끈하지 않지만, 진실하다.

The Afterglow:
"당신의 외장하드에는 무엇이 잠들어 있는가?"
```

**5. Publish (CD + AD)**:
```
시각 요소: 외장하드 사진 (자연광, 먼지 쌓인)
발행일: 다음 월요일 오전 10시
태그: #아카이브 #시간 #WOOHWAHAE
```

### 사례 2: "손님 없는 날" 경험

**1. Capture**:
```
입력: "오늘 예약 취소. 하루종일 책 읽음.
매출 0원이지만 나쁘지 않았다."
```

**2. Connect**:
```
연결:
- 97layer_identity.md → 생계와 직업의 분리
- WOOHWAHAE → Slow, 실용적 미학
- 비교: 일반 미용실 = 회전율 중시
→ 콘텐츠 가능성: 높음 (차별화 명확)
```

**3-5**: (위와 유사 과정 생략)

**최종 콘텐츠**:
```
The Hook:
"비어있는 공간이 주는 선물"

The Manuscript:
[Slow Life Atelier 에세이]

The Afterglow:
"당신은 아무것도 하지 않는 시간을 견딜 수 있는가?"
```

## X. 실패 사례 및 대응

### 실패 사례 1: 과도한 상업성

**문제**:
- CE가 "신규 고객 환영" 콘텐츠 제안
- 할인 언급
- 상업적 톤

**CD 거부 사유**:
> "WOOHWAHAE는 판매하지 않는다. 제안한다."

**수정 방향**:
- 할인 삭제
- "새로운 만남" 철학적 접근
- 조건 없는 환영

### 실패 사례 2: 감정 과잉

**문제**:
- 97layer의 개인적 분노 (정치적 사안)
- 공격적 톤
- 특정 대상 비판

**SA 판단**:
> "개인 감정이지만 브랜드 자산 부적합"

**처리**:
- `knowledge/private/` 보관
- 발행 안 함
- 97layer에게 "보편적 언어로 재진술 필요" 알림

### 실패 사례 3: 트렌드 영합

**문제**:
- AD가 "릴스 챌린지" 제안
- 유행하는 BGM/형식
- WOOHWAHAE 톤과 불일치
