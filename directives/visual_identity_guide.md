# Visual Identity Guide

WOOHWAHAE 브랜드의 시각적 정체성 시스템.
철학을 시각화하는 방법론.

---

## I. 핵심 철학

**"보이지 않는 것을 보이게 하는 것"**

WOOHWAHAE의 시각 체계는:
- 말하지 않고 보여준다
- 설명하지 않고 암시한다
- 채우지 않고 비운다
- 완성하지 않고 열어둔다

**Aesop 벤치마크**:
- 타이포그래피 중심
- 절제된 색상
- 촉각성 (물성)
- 지적 우아함

---

## II. 3대 상징

### 1. 나비 (Butterfly)

**상징**:
- 羽化 (Emergence) - 번데기에서 나비로
- 변화의 과정
- 어둠 속에서의 성장
- 빛을 향한 비상

**이미지**:
- 고치 안의 정적 (before)
- 날개 펼치는 순간 (process)
- 비상 (after - 하지만 끝이 아님)

**사용 원칙**:
- 사실적 나비 이미지 사용 가능 (과학 도감 스타일)
- 추상화 가능 (선 드로잉, 실루엣)
- 색상: 모노크롬 또는 자연색 (형광 금지)
- 배치: 여백 속에 작게 (압도하지 않음)

**금지**:
- ❌ 귀여운 만화체 나비
- ❌ 반짝이는 효과
- ❌ 대칭 완벽주의 (약간 불완전 허용)
- ❌ 과도한 색상

### 2. 블런트 (Blunt Scissors)

**상징**:
- 미용사의 도구
- 절단 = 결정
- 날카로움 = 명확함
- 실용성 = 본질

**이미지**:
- 빈티지 블런트 가위 (새 것 아님)
- 금속 질감 중시
- 선명한 윤곽
- 실루엣 또는 상세 묘사

**사용 원칙**:
- 제품이 아닌 "도구"로서의 이미지
- 사용 흔적 있는 것 (오래된 미학)
- 조명: 측광 (그림자 중시)
- 배경: 단색 또는 목재 질감

**금지**:
- ❌ 반짝이는 신제품 느낌
- ❌ 제품 광고 느낌
- ❌ 과도한 포토샵 보정
- ❌ 무의미한 장식

### 3. 인장 (Ink Stamp)

**상징**:
- 아카이브
- 시간의 증거
- 공식성
- 불가역성 (지울 수 없음)

**이미지**:
- 원형 또는 사각 인장
- "WOOHWAHAE" 텍스트 각인
- 잉크 번짐 효과 허용
- 종이 질감 배경

**사용 원칙**:
- 실제 인장 스캔 (디지털 폰트 아님)
- 완벽하지 않은 찍힘 (불균일 허용)
- 빈티지 잉크 색 (검정, 세피아, 다크 레드)
- 문서 느낌 연출

**금지**:
- ❌ 깔끔한 디지털 렌더링
- ❌ 형광색 잉크
- ❌ 지나치게 선명한 윤곽
- ❌ 무게감 없는 배치

## III. 로고 시스템

### 주 로고 (Primary Logo)

**구성**:
```
WOOHWAHAE
───────────
추상화된 나비 실루엣
```

**특징**:
- 세리프 폰트 (우아함)
- 나비 심볼은 단순화 (3-5개 선)
- 가로 정렬 기본
- 모노크롬

**사용 원칙**:
- 최소 크기: 가로 3cm
- 여백: 로고 높이의 50% 이상
- 배경: 화이트 또는 연한 회색
- 회전/왜곡 금지

### 보조 로고 (Secondary Logo)

**"LIVE FAST DIE NOW WOOHWAHAE" 원형 로고타입**

**의미**:
- 빠름의 시대에 대한 역설적 비판
- "빨리 살고 빨리 죽는" 시대에 저항
- WOOHWAHAE = 대안

**구성**:
- 원형 배치
- 외곽선 타이포그래피
- 중앙 여백 (또는 작은 나비 심볼)

**사용**:
- 인장처럼 사용 (콘텐츠 끝)
- 패키지/스티커
- 서명 대체

### 색상 규정

**Primary Palette**:
- **화이트** (#FFFFFF) - 70%
- **블랙** (#000000) - 20%
- **다크 그레이** (#2C2C2C) - 10%

**Accent Palette** (드물게 사용):
- **세피아** (#704214) - 아카이브 느낌
- **다크 레드** (#5C0F0F) - 인장 느낌
- **아이보리** (#F8F6F0) - 종이 질감

**금지 색상**:
- ❌ 형광색 전체
- ❌ 밝은 파스텔
- ❌ 원색 (빨강, 파랑, 노랑)
- ❌ 그라데이션

## IV. 타이포그래피

### 본문 (Body Text)

**한글**:
- **Primary**: 본명조 / Noto Serif KR
- 무게: Light (300) 또는 Regular (400)
- 줄간격: 1.8-2.0
- 자간: 0%

**영문**:
- **Primary**: Crimson Text / Freight Text
- 무게: Regular (400)
- 줄간격: 1.6-1.8
- 자간: 0%

**원칙**:
- 긴 텍스트 허용 (Aesop 방식)
- 문단 간격 넓게 (1.5배)
- 좌측 정렬 (가운데 정렬 최소화)
- 하이픈 사용 최소화

### 제목 (Heading)

**H1**:
- 폰트: 본명조 Bold / Crimson Pro SemiBold
- 크기: 32-48pt
- 자간: -2%
- 무게감 중시

**H2**:
- 폰트: 본명조 Medium / Crimson Pro Medium
- 크기: 24-32pt
- 자간: 0%

**H3**:
- 폰트: 본명조 Regular / Crimson Pro Regular
- 크기: 18-24pt
- 자간: 0%

**원칙**:
- 제목도 절제 (과장 금지)
- 부제 활용 (설명 대신 암시)
- 숫자 사용 최소화

### 특수 타이포그래피

**인용문 (Quote)**:
- 이탤릭 사용
- 들여쓰기 2em
- 크기: 본문과 동일 또는 +2pt
- 색상: 다크 그레이

**캡션 (Caption)**:
- 폰트: 본고딕 Light / Roboto Light
- 크기: 12-14pt
- 색상: 그레이 (#666666)
- 위치: 이미지 하단 또는 옆

**강조 (Emphasis)**:
- 볼드 최소 사용 (이탤릭 선호)
- 밑줄 사용 금지
- 색상 변경 최소화

## V. 이미지 가이드

### 사진 스타일

**조명**:
- 자연광 선호 (인공 조명 최소)
- 시간: 골든 아워 (아침/저녁)
- 톤: 따뜻한 화이트 (차가운 톤 금지)
- 그림자: 허용 (강한 명암 OK)

**구도**:
- 여백 많이 (60% 이상)
- 중앙 배치보다 3분할
- 수평/수직 중시 (안정감)
- 미니멀리즘

**피사체**:
- 사람: 얼굴 클로즈업 최소, 실루엣/반신 선호
- 공간: 한적함, 정돈됨, 자연광
- 오브제: 빈티지, 사용 흔적, 물성

**색감**:
- 채도 낮춤 (-20%)
- 대비 약간 올림 (+10%)
- 따뜻함 +5-10
- 필터 사용 최소 (VSCO HB1/HB2 정도)

**금지**:
- ❌ 셀카 스타일
- ❌ 과도한 보정 (매끈한 피부)
- ❌ 형광 조명
- ❌ 복잡한 배경
- ❌ 과포화 색상

### 일러스트레이션

**스타일**:
- 선 드로잉 (펜, 잉크)
- 도감 스타일 (과학적 정밀함)
- 미완성 허용 (스케치 느낌)

**금지**:
- ❌ 만화체
- ❌ 3D 렌더링
- ❌ 아이콘 스타일
- ❌ 귀여운 캐릭터

## VI. 레이아웃 원칙

### 그리드 시스템

**기본 그리드**:
- 12컬럼 그리드
- Gutter: 24px
- Margin: 좌우 10%

**콘텐츠 영역**:
- 최대 너비: 800px (읽기 편한 폭)
- 텍스트 블록: 6-8컬럼
- 이미지: 전체 또는 8-10컬럼

**여백 (White Space)**:
- 최소 40% 이상
- 섹션 간격: 80-120px
- 요소 간격: 40-60px

### 구성 원칙

**비대칭 균형**:
- 완벽한 대칭 피하기
- 시각적 무게 분산
- 공간의 호흡

**수직 리듬**:
- 일관된 간격 유지
- 8px 베이스라인
- 모듈화된 간격 (8, 16, 24, 40, 64, 96...)

**계층 구조**:
- 크기보다 간격으로 구분
- 3단계 이하 (H1, H2, Body)
- 명확한 정보 흐름

## VII. 인스타그램 가이드

### 피드 구성

**전체 톤**:
- 모노크롬 + 아이보리
- 균일한 밝기 (밝음 70%)
- 일관된 필터

**게시물 타입**:
1. **텍스트 카드** (40%)
   - 화이트 배경
   - 세리프 타이포그래피
   - The Hook 문장

2. **이미지** (40%)
   - 공간/오브제/질감
   - 여백 많은 구도
   - 자연광

3. **캐러셀** (20%)
   - 긴 텍스트 (The Manuscript)
   - 슬라이드 5-10장
   - 천천히 읽도록 유도

### 캡션 스타일

**구조**:
```
The Hook (첫 줄)

[공백]

The Manuscript (본문 400-800자)

[공백]

The Afterglow (질문으로 끝)

#WOOHWAHAE
```

**원칙**:
- 이모지 사용 최소 (문장 끝에만)
- 해시태그 3개 이하
- 줄바꿈 충분히
- "좋아요 구독" 멘트 금지

### 스토리

**일상 기록**:
- 공간 정리 과정
- 책 읽는 순간
- 자연광 변화
- 침묵의 순간

**원칙**:
- 과도한 스티커 금지
- 텍스트 최소화
- 음악: 앰비언트/클래식
- 시간: 24시간 후 삭제 (하이라이트 최소)

## IX. 공간 디자인 (오프라인)

### 미용실 (현재)

**색상**:
- 벽: 화이트 또는 아이보리
- 바닥: 내추럴 우드 또는 콘크리트
- 가구: 다크 우드 또는 블랙 메탈

**조명**:
- 자연광 최대 (큰 창)
- 간접 조명 (따뜻한 화이트 3000K)
- 조명기구: 미니멀 (펜던트 1-2개)

**가구**:
- 빈티지 선호 (새 것 금지)
- 사용 흔적 허용
- 미니멀 디자인
- 낮은 가구 (시야 개방)

**오브제**:
- 식물 (큰 것 1-2개, 작은 것 여러 개)
- 책 (인문학/사진집)
- 세라믹 (화기, 오브제)
- 패브릭 (린넨, 코튼)

**금지**:
- ❌ 네온사인
- ❌ 화려한 인테리어
- ❌ 거울 많이 (필요한 것만)
- ❌ 제품 진열 과다

### 확장 공간 (미래)

**서점/카페**:
- Aesop 매장 철학 참고
- 촉각성 중시 (나무, 종이, 천)
- 오래 머무르도록 (편한 의자)
- 조용함 (BGM 앰비언트)

**갤러리/전시**:
- 화이트 큐브
- 작품 간격 넓게
- 조명: 스포트라이트
- 설명 최소화 (QR 코드)

## XIII. 시즌별 변주

계절에 따른 시각 체계 변주 (철학은 불변).

### 봄 (3-5월)

**색감**: +5 밝기, +3 따뜻함
**피사체**: 꽃 (단, 과장 금지), 새싹, 빛
**분위기**: 부드러움

### 여름 (6-8월)

**색감**: +10 대비, 0 색온도
**피사체**: 그림자, 물, 여백
**분위기**: 간결함

### 가을 (9-11월)

**색감**: +5 채도, +5 따뜻함
**피사체**: 낙엽, 목재, 세피아
**분위기**: 깊이

### 겨울 (12-2월)

**색감**: -5 채도, -3 밝기
**피사체**: 흑백, 실루엣, 정적
**분위기**: 침묵

## XIV. 체크리스트

모든 비주얼 작업 전 확인 사항.

### 로고 사용

- [ ] 최소 크기 준수 (가로 3cm)
- [ ] 여백 충분 (로고 높이의 50% 이상)
- [ ] 화이트 배경 또는 연한 회색
- [ ] 회전/왜곡 없음

### 타이포그래피

- [ ] 세리프 폰트 사용
- [ ] 줄간격 1.8 이상
- [ ] 좌측 정렬
- [ ] 긴 텍스트 허용

### 이미지

- [ ] 자연광
- [ ] 여백 60% 이상
- [ ] 채도 낮춤
- [ ] 과도한 보정 없음

### 색상

- [ ] 모노크롬 기본
- [ ] 형광색 없음
- [ ] 그라데이션 없음

### 톤

- [ ] WOOHWAHAE 철학과 일치
- [ ] 과장 없음
- [ ] 절제됨
- [ ] 깊이 있음
