# Anti-Algorithm Protocol

알고리즘 저항 프로토콜.
휘발적 콘텐츠, 자극, 양극화에 대한 대응 전략.

---

## I. 문제 인식

### 97layer의 알고리즘 비판

**원문**:
> "촘촘히 짜여진 알고리즘의 양극화, 자극적이고 알맹이 없는 휘발적 콘텐츠, 세 줄 요약, 소모적 가십거리... 편안하지만 생각 안 해도 됨."

**3대 문제**:

1. **휘발성** (Volatility)
   - 빠르게 소비, 빠르게 잊힘
   - 3초 집중, 릴스/쇼츠 중독
   - 깊이 없는 소비

2. **자극성** (Stimulation)
   - 클릭베이트, 과장된 썸네일
   - 감정 과잉 (분노, 선정)
   - 알고리즘이 강요하는 톤

3. **양극화** (Polarization)
   - 찬반 강요
   - 빠른 판단 유도
   - 뉘앙스 삭제

**97layer의 대응**:
> "인간의 강력한 고유함은 사유하는 능력. 한 발 물러서서 다각적으로 바라보는 감각. 열린 마음과 냉철한 이성."

---

## II. 프로토콜 정의

**Anti-Algorithm이란**:
- 알고리즘 최적화 거부
- 조회수/좋아요 추구 거부
- 빠른 소비 거부
- 대신: 깊이, 축적, 느림

**목표**:
- WOOHWAHAE = 알고리즘의 대안
- Slow by design
- 축적되는 콘텐츠
- 소수의 깊은 독자

---

## III. 3대 거부 원칙

### 1. 휘발 거부 (Anti-Volatility)

#### 문제
- 3초 집중 시대
- 릴스/쇼츠 = 즉각적 소비 후 망각
- "다음, 다음, 다음" 무한 스크롤

#### 대응
**긴 텍스트 허용**:
- 인스타그램 캡션: 400-800자
- 블로그/웹: 1500-3000자
- 읽는 시간 표시 ("5분 읽기")
- Aesop 방식: 긴 호흡

**슬로우 포맷**:
- 캐러셀: 10장 이상 (천천히 읽도록)
- 롱폼 영상: 5-15분 (릴스 거부)
- 정적 이미지 (동영상 최소)

**상호참조**:
- 과거 콘텐츠 링크
- "지난 글 참조"
- 시리즈 콘텐츠 (연결성)
- 타임라인 구축

**아카이브 구축**:
- 모든 콘텐츠 웹사이트 보관 (미래)
- 삭제 금지
- 날짜 태그 (시간 축적 가시화)

#### 실천 예시

**Bad** (휘발적):
```
[3초 릴스]
"헤어 꿀팁! 💇‍♀️✨"
[빠른 컷 편집]
[유행 BGM]
```

**Good** (반-휘발):
```
[캐러셀 10장]

The Hook:
"시간은 흘러내리지만, 기록은 남는다"

[이미지 1-2: 공간 사진]
[이미지 3-8: 긴 텍스트, 천천히 읽도록]
[이미지 9: The Afterglow 질문]
[이미지 10: 과거 콘텐츠 참조]

읽는 시간: 5분
```

### 2. 자극 거부 (Anti-Stimulation)

#### 문제
- 클릭베이트 제목
- 과장된 썸네일
- 감정 과잉 (😱😍😭🔥)
- "놀라운", "충격적", "최고의"

#### 대응

**절제된 언어**:
- 형용사 최소화
- 사실 중심
- 감정 단어 배제 ("사랑해요", "대박")
- Aesop 톤 유지

**단조로운 시각**:
- 모노크롬
- 단순 배경
- 썸네일 = 본문 첫 이미지 (따로 제작 안 함)
- 화려한 효과 금지

**낮은 목소리**:
- 외치지 않음
- 강요하지 않음
- 제안만 함
- "좋아요 구독" 멘트 금지

#### 실천 예시

**Bad** (자극적):
```
제목: "😱충격😱 이 방법 몰랐다면 손해! 지금 바로👆"
썸네일: 과장된 표정, 화살표, 강조 텍스트
내용: "대박", "완전", "진짜 좋아요"
```

**Good** (반-자극):
```
제목: "공간과 시간에 대한 단상"
썸네일: 여백 많은 공간 사진 (텍스트 최소)
내용: "시간은 흐른다. 우리는 기록한다."
```

---

### 3. 양극화 거부 (Anti-Polarization)

#### 문제
- 찬반 강요 ("동의하면 좋아요")
- 빠른 판단 유도
- 복잡함 삭제 ("3줄 요약")
- 타인 비난

#### 대응

**회색 지대 허용**:
- 찬반 강요 안 함
- "이렇게도, 저렇게도" 뉘앙스
- 결론 안 내림 (질문으로 끝)

**복잡함 유지**:
- "3줄 요약" 거부
- 맥락 충분히 설명
- 양가감정 표현 (협업에 대한)
- 단순화 거부

**비판보다 대안**:
- 특정 브랜드 저격 안 함
- "이건 나쁘다" 대신 "이건 대안"
- 97layer 방식 제시
- 타인 비방 금지

#### 실천 예시

**Bad** (양극화):
```
"요즘 미용실은 다 장사꾼!
진짜 실력 있는 곳은 여기뿐!
동의하면 좋아요👍"
```

**Good** (반-양극화):
```
"생계와 직업을 분리한다는 것.
쉽지 않다. 모순도 있다.
하지만 시도해본다.

당신은 어떻게 일하는가?"
```

## IV. 알고리즘 최적화 거부 목록

### 인스타그램 알고리즘이 요구하는 것

1. **릴스 우선 노출** → 거부 (롱폼 고수)
2. **일 2-3회 발행** → 거부 (주 1-2회)
3. **3초 내 후킹** → 거부 (천천히 읽도록)
4. **해시태그 30개** → 거부 (3개 이하)
5. **스토리 활발** → 최소 (하루 1개 이하)
6. **댓글 유도** → 강요 안 함
7. **좋아요 구독 멘트** → 절대 안 함
8. **트렌딩 오디오** → 사용 안 함 (앰비언트만)
9. **콜라보/태그** → 선택적 (결 맞는 것만)
10. **라이브** → 안 함

### 대신 하는 것

1. **긴 캡션** (알고리즘 불리)
2. **정적 이미지** (동영상 밀림)
3. **느린 발행** (일관성만 유지)
4. **최소 해시태그**
5. **조용한 운영**
6. **자연스러운 대화** (댓글 답변 선택적)
7. **무언의 초대** (강요 없음)
8. **자체 제작 BGM** (또는 무음)
9. **선택적 협업**
10. **비공개 소통** (DM, 예약 상담)

**결과**:
- 조회수 낮음 (예상됨)
- 팔로워 적음 (예상됨)
- 하지만 깊은 연결
- 충성 고객 (게스트)

## V. 콘텐츠 타입별 대응

### 1. 텍스트 콘텐츠

**알고리즘 방식**:
- 3줄 요약
- 핵심만 빠르게
- 이모티콘 많이
- 쉬운 언어

**WOOHWAHAE 방식**:
- 800자 에세이
- 과정까지 천천히
- 이모티콘 최소
- 지적 언어 (하지만 접근 가능)

### 2. 이미지 콘텐츠

**알고리즘 방식**:
- 밝고 화려
- 얼굴 클로즈업
- Before/After
- 텍스트 오버레이 많이

**WOOHWAHAE 방식**:
- 모노톤, 여백
- 공간/오브제 중심
- 과정 중시
- 텍스트 최소 (타이포그래피만)

### 3. 영상 콘텐츠

**알고리즘 방식**:
- 15-30초 릴스
- 빠른 컷
- 유행 BGM
- 자막 많이

**WOOHWAHAE 방식**:
- 5-15분 롱폼
- 느린 컷 (5초+ 유지)
- 앰비언트/재즈 또는 무음
- 자막 최소 (필요한 것만)

## VI. 측정 지표 재정의

### 거부하는 지표 (Vanity Metrics)

- ❌ 조회수
- ❌ 좋아요 수
- ❌ 팔로워 수
- ❌ 리치 (도달 범위)
- ❌ 바이럴 여부

**이유**:
> "숫자로 표현할 수 없는 것들에 시간을 썼다" (97layer)

### 추구하는 지표 (Meaningful Metrics)

- ✅ 재방문율 (고객 충성도)
- ✅ 평균 읽기 시간 (깊이)
- ✅ DM 깊이 (진정한 대화)
- ✅ 예약 상담 시간 (1시간+)
- ✅ 아카이브 축적 (연간 50-100개)
- ✅ 상호참조 빈도 (과거 콘텐츠 연결)

**목표**:
- 100명의 팬 > 10,000명의 구경꾼
- 1시간 대화 > 1,000개 좋아요
- 1년 축적 > 1개 바이럴

## VII. 에이전트 역할

### SA (Strategy Analyst)

**DO**:
- 알고리즘 트렌드 모니터링 (피하기 위해)
- 경쟁사 알고리즘 추종 분석
- WOOHWAHAE 차별성 리포트

**DON'T**:
- "이 트렌드 따라가면 좋겠다" 제안
- 조회수 증가 전략 제안
- 바이럴 콘텐츠 아이디어

### CE (Chief Editor)

**DO**:
- 긴 텍스트 작성 (800자+)
- 절제된 언어 유지
- 깊이 있는 서사

**DON'T**:
- "좀 더 자극적으로" 수정
- "3줄 요약 추가" 제안
- "이모티콘 넣으면" 제안

### CD (Creative Director)

**DO**:
- 알고리즘 저항 원칙 수호
- "조회수 낮아도 괜찮다" 확신
- 장기 비전 유지

**DON'T**:
- 단기 성과 압박
- 알고리즘 타협 제안
- "이번만" 예외 허용

### AD (Art Director)

**DO**:
- 모노톤 시각 유지
- 여백 많은 구도
- 느린 영상 (롱폼)

**DON'T**:
- "화려하게" 수정
- 릴스 포맷 제안
- 트렌딩 스타일 모방

## VIII. 위기 대응

### 위기 1: "조회수가 너무 낮아요"

**97layer 불안**:
> "아무도 안 보는 것 같은데... 의미 있을까?"

**CD 대응**:
```
예상된 결과입니다.

Aesop 인스타그램: 팔로워 100만+, 게시물 평균 좋아요 3-5천
→ 참여율 0.3-0.5% (매우 낮음)

하지만 Aesop은 전 세계 수백 개 매장 운영.
조회수가 목적이 아니기 때문.

WOOHWAHAE도 마찬가지.
소수의 깊은 연결 > 다수의 얕은 노출
```

**대안 제시**:
- 예약 상담 신청 증가 체크
- DM 깊이 분석 (짧은 칭찬 vs. 긴 대화)
- 재방문 고객 비율
- 입소문 (오프라인 추천)

### 위기 2: "경쟁사는 릴스로 성공하는데"

**97layer 불안**:
> "다른 미용실은 릴스로 고객 많이 오는데..."

**SA 대응**:
```
경쟁사 분석:

타 미용실 A: 릴스 바이럴 → 신규 고객 100명/월
하지만 재방문율 30%, 가격 경쟁, 소진

타 미용실 B: 조용한 운영 → 신규 고객 10명/월
하지만 재방문율 80%, 충성 고객, 지속 가능

97layer는 B를 선택했습니다.
생계와 직업의 분리, 본질 추구, 느린 성장 신뢰.
```

### 위기 3: "알고리즘이 계속 변해요"

**TD 대응**:
```
알고리즘은 계속 변합니다.
하지만 우리는 변하지 않습니다.

2015년: 사진 중심
2018년: 스토리 중심
2021년: 릴스 중심
2024년: AI 추천

우리는 2025년부터:
긴 텍스트, 느린 영상, 깊이 축적.

10년 후에도 같을 것입니다.
알고리즘이 어떻게 변하든.
```

## IX. 성공 사례 (타 브랜드)

### 1. Aesop

**전략**:
- 릴스 거의 없음 (연 10개 이하)
- 긴 캡션 (500-1000자)
- 정적 이미지 중심
- 조회수 낮음

**결과**:
- 글로벌 브랜드
- 고가 정책 유지
- 충성 고객 확보

### 2. Kinfolk Magazine

**전략**:
- SNS 최소 활동
- 계간지 중심 (느림)
- 긴 에세이 (5000자+)
- 트렌드 무시

**결과**:
- 컬트 브랜드 지위
- 독자 충성도 높음
- 출판 + 카페 + 이벤트 확장

### 3. The Monocle

**전략**:
- 라디오 (Monocle 24) 중심
- 긴 인터뷰
- 월간지
- SNS 보조 수단

**결과**:
- 글로벌 미디어 브랜드
- 오프라인 매장 운영
- 지적 권위 확보

## X. 장기 비전

### 1년 차

**목표**:
- 알고리즘 저항 습관화
- 조회수 무시 훈련
- 긴 호흡 콘텐츠 생산

**지표**:
- 주 1-2회 발행 (52-104개/년)
- 평균 800자 유지
- 릴스 0개

### 3년 차

**목표**:
- "알고리즘 대안" 브랜드 인식
- 소수 충성 고객 확보
- 웹사이트 아카이브 완성

**지표**:
- 150-300개 콘텐츠 축적
- 재방문율 70%+
- 입소문 비율 증가

### 5년 차

**목표**:
- WOOHWAHAE = Slow Life 대명사
- 출판 (에세이집)
- 알고리즘 저항 방법론 공유

**지표**:
