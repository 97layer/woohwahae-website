---

## I. 구조 (The Architecture)

```
97layer          — 인간. 사유하고 기록하는 사람.
    ↓
WOOHWAHAE        — 브랜드. 그 사유의 외부 발현체.
    ↓
97layerOS        — 시스템. 발현을 자동화하는 순환 체계(THE CYCLE).
```

세 층은 독립적이지 않다. 97layer의 내면이 WOOHWAHAE가 되고, WOOHWAHAE의 축적이 97layerOS를 통해 순환한다.

## II. 97layer — The Individual (개인)

### 근본 문제 5가지와 OS의 해결

| 문제 | 해결 |
|---|---|
| 완벽주의 마비 → 시도 못함 | "일단 저장" — 신호 수신 즉시 저장, 판단 보류 |
| 휘발의 반복 → 파편 축적 안됨 | council_room.md append + INTELLIGENCE_QUANTA.md 상태 유지 |
| 사유→행동 어려움 | 텔레그램 한 줄 → 큐 → 에이전트 처리 (개입 최소화) |
| 파편의 무의미 | Signal → NotebookLM → RAG → 연결 발견 |
| 알고리즘 압력 | 시스템이 축적, 사용자는 브랜딩에만 집중 |

### 정체성

- **슬로우 라이프 실천자**: 속도보다 방향, 효율보다 본질.
- **불완전함의 수용**: 72시간 규칙 — 완벽한 지연보다 불완전한 완료.
- **기록하는 사람**: "기억은 흐려지지만, 기록은 남는다."
- **공간**: 반지하 8평 원룸 — 좁지만 온전히 나만의 속도로 살아가는 공간.

## III. WOOHWAHAE — The Brand (브랜드)

### 핵심 선언

> **"Remove the Noise, Reveal the Essence"**
> 소음을 제거하고, 본질을 드러낸다.

WOOHWAHAE는 헤어 아틀리에이자 슬로우 라이프의 제1 발현체다. 빠른 트렌드가 아닌 개인의 본질적 아름다움을 찾는 공간. 기술보다 철학이 먼저인 곳.

### 5 Pillars

1. **진정성 (Authenticity)**: 가면 없는 대화, 취약함의 수용.
2. **실용성 (Practicality)**: Wash & Go 철학, 지속 가능한 관리.
3. **우아함 (Elegance)**: 절제된 표현, 여백의 미학, 조용한 고급스러움.
4. **정밀함 (Precision)**: 구조적 사고, 기반의 견고함.
5. **혁신 (Innovation)**: 관습의 파괴, Anti-Uniformity, 새로운 정의.

### Forbidden

- ❌ 알고리즘 추종 — 숫자보다 한 사람의 진심 어린 공감을 우선함
- ❌ 과도한 장식 — 본질을 가리는 모든 수식어와 복잡성 배제
- ❌ 무리 짓기 — 고독의 가치를 훼손하는 소모적 사교 경계
- ❌ 속도 압박 — 알고리즘의 업로드 주기에 맞추는 행위 금지

## IV. Visual & Communication Identity

### 시각 원칙

- **여백**: 화면의 60% 이상 여백 — 시각적 숨통.
- **톤**: Monochrome(흑백) 기본, Deep Navy + Warm Gray 절제 사용.
- **빛**: 자연광과 그림자 — 시간의 흔적.
- **서체**: 국문(본명조 — 정갈한 클래식), 영문(Crimson Text — 우아한 세리프).
- **기준점**: Aesop 브랜드 시각 언어 70%+ 수준.

### 콘텐츠 목소리

- **톤**: 사려 깊고(Thoughtful), 진정성 있으며(Authentic), 정밀함(Precise).
- **길이**: 캡션 50-100단어. 과잉 설명 금지.
- **금지 키워드**: "대박", "꿀팁", "핫", "트렌디", 과장 형용사 일체.
- **허용 키워드**: 본질, 기록, 순간, 흔적, 절제, 고요함, 온전함.

### 서사 구조

```
The Hook    — 정의, 역설, 질문을 통한 명료한 선언
The Story   — 개인 경험(Opening) → 보편 전환(Bridge) → 핵심(Core) → 여운(Closing)
```

## V. THE CYCLE — 순환 체계

97layerOS가 구현하는 브랜드 생산 순환:

```
입력 (텔레그램/신호)
  ↓
저장 (knowledge/signals/)
  ↓
연결 (NotebookLM RAG — 패턴 발견)
  ↓
생성 (SA → CE → AD → CD 에이전트 체인)
  ↓
발행 (WOOHWAHAE 콘텐츠)
  ↓
다시 입력 (반응, 인사이트 → 새 신호)
```

사용자가 할 일: 텔레그램에 한 줄 입력. 나머지는 THE CYCLE이 처리한다.
